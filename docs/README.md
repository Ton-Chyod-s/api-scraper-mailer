# Documentação

Esta pasta é servida com Docsify.

Para abrir localmente:

```bash
npm run docs
```

## Guias

- `SETUP.md`: instalação, variáveis de ambiente, banco e comandos
- `DOCKER.md`: como rodar com Docker e Docker Compose (dev e prod)
- `ENV.md`: consulta rápida de variáveis de ambiente
- `API.md`: endpoints disponíveis e exemplos
- `NEW_FEATURE.md`: passo a passo para criar uma nova feature seguindo a arquitetura do projeto
- `ARCHITECTURE.md`: camadas, regras de dependência e fluxo das requests
- `QUALITY.md`: lint, prettier, testes e validações
- `DEPLOYMENT.md`: build, execução em produção e dicas de proxy
- `SECURITY.md`: práticas de segurança, autenticação e recomendações
- `CONTRIBUTING.md`: padrão de contribuição, PRs e checklist
- `TROUBLESHOOTING.md`: erros comuns e como resolver
